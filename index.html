Quickstart

npm install @actionpackd/sdk

Basic Example

import { serverCompose } from '@actionpackd/sdk';

const schema = {
  type: 'object',
  properties: {
    summary: { type: 'string' },
    sentiment: { type: 'string', enum: ['positive', 'neutral', 'negative'] }
  }
};

const response = await serverCompose(
  "Analyze this tweet: 'Just launched our new AI SDK!'",
  schema
);

Documentation

Why Schema-first?

Schema-first development ensures type safety and predictable AI responses. Define your expected output structure and let Actionpackd handle the validation.

// Your schema defines the shape of AI responses
const schema = z.object({
  title: z.string(),
  categories: z.array(z.string())
});
